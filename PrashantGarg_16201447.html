COMP41680 ASSIGNMENT 1: Data Collection & Preparation 2016/2017¶

The project analyses the attributes such as number of hotels, pubs, museums, shopping centers, event and temperature for the most visited places in the world. To do so, the data had to be collected from various sources. As the first step of the data collection the web scrapping technique is used to extract most visited places in world.

The names of the cities and their countries are obtained from web scrapping. This data extracted acts as input to the Google's Geocode API to find the geographic coordinates of these cities. This coordinate information assists in finding out the different attributes of the city such as the current temperature, number of hotel, bars, museums and shopping centers present in the city. Also, the number of different event to be held in the current month are extracted.

There are total four Web APIs used for this assignment:

  1. Geocode API - The Geocode API converts the addresses (like a street address, cities names) into geographic coordinates (like latitude and longitude).
    • Standard Usage Limit of Geocode API (Free Users)
      2,500 free requests per day
      50 requests per second
      500 requests per hour
    • No. of Hits to Geocode API in the assignment = 100
    • API End Point: https://maps.googleapis.com/maps/api/geocode/

  2. Weather API of OpenWeatherMap - Temperature of the Cities is extracted using the Weather API of OpenWeatherMap
    • Standard Usage Limit of Weather API of OpenWeatherMap(Free Account)
      60 requests per minute
    • No. of Hits to OpenWeatherMap API in the assignment= 100
    • API End Point: http://api.openweathermap.org/

  3. FourSquare Web API - The information of the Bars, Hotels, Shopping Centers and Museums is extracted from FourSquare Web API
    • Standard Usage Limit of FourSquare Web API(Free Account)
      5000 requests per hour (User less)
    • No. of Hits to FourSquare API in the assignment= 400
    • API End Point: https://api.foursquare.com/

  4. Eventbrite Web API - The information of the Events to be held (for the current month) is extracted from Eventbrite Web API
    • Standard Usage Limit of Eventbrite Web API(Free Account)
      1,000 calls per hour
      24,000 calls per day
    • No. of Hits to Events API in the assignment= 100
    • API End Point: https://api.foursquare.com/

# This method fetches the latitude and longitude of the cities retrieved from Step_1

def fetchGeoInfoCities(step1Data):
    
    #Empty Dataframe with attributes such as Rank, City, Country, Tourist, Latitude, Longitude is created
    dataFrameExtractedGeoInfo = pd.DataFrame({"Rank":[],"City":[],"Country":[],"Tourists":[],"Latitude":[],"Longitude":[]})
    
    
    print('\nFetching the Latitude and Longitude for the Cities')
    try:
        
        # For every city in the list of cities
        for singleDataEntry in step1Data:
            rank = singleDataEntry[0] # Rank of the city from step_1
            city = singleDataEntry[1].replace(" ","") # City name from step_1
            country = singleDataEntry[2].replace(" ","") # Country in which the city is from step_1
            tourists = singleDataEntry[3] # number of tourists travelling to that city from step_1
            
            #Fetching the Lattitude and Longitude Information from Google Map API
            linkLocation = geocodeAPIEndPoint + 'json?address=' + city + ','+country+"&key=" + api_key_geocode
            responseGeocodeAPI = req.urlopen(linkLocation)
            dataGeocodeAPI = json.load(responseGeocodeAPI)
            
            # Parsing the json obtained by the API hit and extracting Latitude and Longitude from it
            for item in dataGeocodeAPI['results']:
                lat = item['geometry']['location']['lat']
                lon = item['geometry']['location']['lng']
                
                # The Latitude and Longitude retrieved are appended to the data retrieved in step_1
                df = pd.DataFrame({"Rank":[rank],"City":[singleDataEntry[1]],"Country":[singleDataEntry[2]],"Tourists":[singleDataEntry[3]],"Latitude":[lat],"Longitude":[lon]})
            
            # The details of cities is extracted one at a time and appended to the main dataframe
            dataFrameExtractedGeoInfo = pd.concat([dataFrameExtractedGeoInfo,df])
            
            #  after every city's geographic info is retrieved the progress of the Step_2 operation is updated 
            percentageGeo.update()
            
        #The DataFrame containing the geographic coordinates of all the cities is returned
        return dataFrameExtractedGeoInfo
    except Exception as e:
        print("Got error, while retrieving Latitude and Longitude " , e)
        return None

# Method Name: [generateCSVGeocodeAPI]
# This method fetches the most travelled cities in the world , it checks the status of table 
#  migration before and after its real execution
# @param data represents the data to be stored in the csv file
# @param step denotes step in which the data was extracted from Web API
# @param status nameOfCSV denotes the with which the CSV  file would be stored
# @return True if the CSV is generated successfully otherwise returns False
# 
def generateCSVGeocodeAPI(data, step, nameOfCSV):
    try:
        if step is 'geocode':
            #Generating CSV for Step_2 Data Collection Data
            data.to_csv(nameOfCSV, header=True, columns=['Rank','City','Country','Tourists', 'Latitude','Longitude'], index= False)
        elif step is 'tourAttr':
             #Generating CSV for Step_3 Data Collection Data
            data.to_csv(nameOfCSV, header=True, columns=['Rank','City','Country','Tourists', 'Temperature','Hotels','Bars', 'Museums','Shopping Centers', 'Events'], index=False)
        elif step is 'scrap':
             #Generating CSV for Step_1 Data Collection Data
            data.to_csv(nameOfCSV,index = False)
        return True
    except Exception as e:
        print("Some error while generating the CSV : " , nameOfCSV , e)
        return False

Conclusion & Recommendation¶

The most visited cities in the world were analyzed on the basis of different attributes such as Temperature, number of hotels, bars, museums, shopping centers and number of events scheduled for the current month. It was found that number of hotels in the city didn't have any impact on attracting the tourists. Also, the temperature was not the reason for the tourists to choose their destination.

However, it was seen that more number of Shopping centers, Bars and Museums often attracted more tourists.

It was seen that, China has 9 of the total most visited cities. This was reason which led the maximum tourists travelling to China (43.7 million).

It was also observed that London was the best overall, when comparing the cities on the basis of the Tour Based Attributes.

The average values of the Tour Based Attributes were really on the higher side, which led me to choose Median to fill the missing values if any.

Next, are the recommendation of ten cities to visit to the tourists on the basis of the Tour Based Attributes which are Shopping Centers, Events, Bars, Museums and Hotels.

Since, number of Hotels have no correlation with tourists travelling to a city. I have just considered the number of hotels to be more than the first quartile, for all the other attributes they should be more than the mid values to be considered.
